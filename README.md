# TodoApi

API RESTful  **Minimal API** y **SQLite**.


probando .NET 8 - Entity Framework Core- SQLite


Endpoints

GET `/api/todo` Trae una lista 
POST  `/api/todo` Crea 
GET `/api/todo/{id}` Trae por id
PUT `/api/todo/{id}` Actualiza por id
DELETE `/api/todo/{id}` Elimina por id
